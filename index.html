Game Mania - Angular Material

Criado : 21/12/2021
Por : ReaperClown

O intuito do site Game Mania é para fins acadêmicos a fim de colocar em prática os conteúdos sobre o framework Angular Material.
O site é puramente fictício e nenhum produto, preço, ou informação (financeira, legal, empresarial, etc) são verdadeiros, logo não devem ser considerados.

Recursos


O site conta com o framework Bootstrap v3.3.4 com suas tags HTML5 válidas. O site segue o layout de quatro colunas do padrão grid.

Primeiros passos


Preparação

  1. Tenha certeza de que o Node.js está instalado.
  2. Abra o terminal no editor de escolha e navegue até o diretório src.
  3. Execute o commando npm install. Este comando baixará todas as dependências exigidas pelo site, definidas no arquivo package.json.
  4. Então execute o comando npm install -g @angular/cli para instalar a Angular CLI (caso não esteja instalada).

Executando o projeto

Para iniciar o servidor HTTP local nativo do angular, com live-reload, execute o commando: 

ng serve

Para criar a build para o cenário de produção, execute:

ng build --prod

Após a execução e então a compilação, o projeto compilado estará localizado na pasta dist, na raiz do projeto.

Para uma lista completa dos commandos da Angular CLI, consulte a documentação oficial.
